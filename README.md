# README

<h2>Grapptitude</h2>

This is a Ruby on Rails API/React-Redux front end free application for sharing things you are grateful for.

<h2>Installation</h2>

* Git clone into this repo
* CD grapptitude
* Bundle install
* Run rails db:create, rails db:migrate & rails db:seed
* CD client
* Run npm install
* Run rake start
